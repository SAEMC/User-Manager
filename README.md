# SAEMC User Manager - Ver_1_0

### 준비 사항

- Git 설치
```
$ (sudo) apt-get update && \
  (sudo) apt-get install -y git
```

- SAEMC User Manager 프로젝트 다운로드
```
$ git clone https://github.com/SAEMC/User-Manager.git
```

---

### 사용 방법

- `./user_manager.sh` 실행
```
$ cd ./User-Manager && ./user_manager.sh
```
