# SAEMC User Manager

### 준비 사항

- Git 설치
```bash
$ (sudo) apt-get update && \
  (sudo) apt-get install -y git
```

- SAEMC User Manager 프로젝트 다운로드
```bash
$ git clone https://github.com/SAEMC/User-Manager
```

---

### 사용 방법

- `./user_manager.sh` 실행
```bash
$ cd ./User-Manager && ./user_manager.sh
```
